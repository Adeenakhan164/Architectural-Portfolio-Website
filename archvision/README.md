## 📄 Project Description

**ArchVision Studio** is a professional architectural firm website that showcases services, projects, and a company profile. It simulates a real-world architecture firm's online presence where users can explore projects, view services, learn about the company, and contact the team.

---

## 🌐 Live Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Hero section, featured projects, services preview, testimonials |
| About | `about.html` | Company story, team profiles, mission & vision |
| Services | `services.html` | Full service cards + 5-step design process |
| Projects | `projects.html` | Filterable portfolio gallery |
| Contact | `contact.html` | Form with validation + Google Maps |

---

## ✅ Features Implemented

### Functional Features
- ✅ Responsive Navigation Bar (hamburger on mobile)
- ✅ Dark / Light Mode Toggle (saved in localStorage)
- ✅ Animated Counter Statistics
- ✅ Auto-rotating Testimonials Slider
- ✅ Project Category Filter (JavaScript)
- ✅ Contact Form with Full Validation
- ✅ Google Maps Integration
- ✅ Scroll Reveal Animations
- ✅ Back-to-Top Button
- ✅ Smooth Scrolling

### Non-Functional Features
- ✅ Fully Responsive (mobile, tablet, desktop)
- ✅ Clean, semantic HTML5 structure
- ✅ Fast-loading images (lazy loading)
- ✅ Accessible (ARIA labels, keyboard focus)
- ✅ Reduced motion support

## 🚀 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YourUsername/archvision-studio.git
   ```

2. **Open the project:**
   ```bash
   cd archvision-studio
   ```

3. **Run in browser:**
   - Simply open `index.html` in any modern browser
   - No server or build step required
